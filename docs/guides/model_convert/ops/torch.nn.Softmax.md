## torch.nn.Softmax
### [torch.nn.Softmax](https://pytorch.org/docs/stable/generated/torch.nn.Softmax.html?highlight=nn+softmax#torch.nn.Softmax)

```python
torch.nn.Softmax(dim=None)
```

paddle.nn.Softmax(axis=- 1,

| PyTorch       | PaddlePaddle | 备注                                                   |
