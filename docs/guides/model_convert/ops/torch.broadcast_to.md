## torch.broadcast_to

```python
torch.broadcast_to(input, 
                    shape)
```

### [paddle.broadcast_to](https://www.paddlepaddle.org.cn/documentation/docs/zh/api/paddle/broadcast_to_cn.html#broadcast-to)

paddle.broadcast_to(x, 
                    shape, 
                    name=None)

| PyTorch       | PaddlePaddle | 备注                                                   |
