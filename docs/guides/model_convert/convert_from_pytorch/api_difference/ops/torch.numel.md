### [paddle.Tensor.size](https://www.paddlepaddle.org.cn/documentation/docs/zh/develop/api/paddle/Tensor_cn.html#size)

```python
paddle.Tensor.size
```

### 转写示例
```python
# PyTorch 写法
torch.numel(x)

# Paddle 写法
x.size
```
