# 飞桨对昆仑芯 2 代芯片的支持

飞桨自 2.4rc 版本起支持在昆仑芯 2 代芯片上（R200，R300，R200-8F，R200-8FS，RG800）运行，经验证的模型训练的支持情况如下：
