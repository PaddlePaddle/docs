####################
昇腾 NPU 芯片运行飞桨
####################

华为昇腾 910（Ascend 910）是一款具有超高算力的 AI 处理器。Paddle NPU 版当前可以支持在华为鲲鹏 CPU 与昇腾 NPU 上进行模型训练与推理。

参考以下内容可快速了解和体验在昇腾芯片上运行飞桨：

- `飞桨框架昇腾 NPU 版训练示例 <./train_example_cn.html>`_ : 飞桨框架昇腾 NPU 版训练示例
