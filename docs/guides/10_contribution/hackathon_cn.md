# 活动介绍

2021 PaddlePaddle Hackathon飞桨黑客马拉松正式上线，本次活动是面向全球开发者的深度学习领域编程活动，鼓励开发者了解与参与深度学习开源项目。

本次飞桨黑客马拉松由飞桨联合深度学习技术及应用国家工程实验室主办，由线上任务认领以及线下 Coding Party 两个部分组成。

线上任务认领环节，开发者可个人或组队报名并认领相应任务，根据任务要求完成相应开发，参与项目共建，任务由飞桨框架、飞桨家族、飞桨生态、开放性任务四个专区组成，并有三个难度梯队，任你选择。线下 Coding Party环节，开发者现场coding，组队挑战。

对深度学习感兴趣你，快来参加活动吧，更有机会获得现金奖励以及来自百度和开源社区技术大咖们的指导与经验分享，更多奖励等你来拿。


## 一、活动流程

1、本次活动整体流程，具体说明如下：

- 1、活动报名：点击**此处**（问卷星链接）填写相关信息，完成活动报名；
- 2、报名成功后，你可以进入百度飞桨黑客松QQ交流群（群号：343734965），所有活动相关信息都会在群中及时同步；
- 3、**任务认领以及开放性任务提交**
- - **任务认领：**在 GitHub 的 **Pinned ISSUE** ，按格式回复你的信息以及要认领的任务，**完成任务认领**；
    o  **开放性任务提交：**如果你对飞桨框架以及飞桨家族相关项目有更好的功能建议，也可以通过在相应仓库下提交 ISSUE 的形式，提交开放性任务，并在在 GitHub 的 **Pinned ISSUE**，按格式回复你的信息以及你挑的任务，评审会将在提交后3个工作日内反馈你任务是否通过审核以及对应难度；

- 4、完成任务认领或开放性任务通过审核后，

- - **容易&中等任务**：按照任务要求进行开发即可

  - **困难任务：**

  - - 按照任务要求在认领的任务 issue 下新建 RFC，并对任务进行技术方案设计；
    - **10****月12日前**提交你对该任务的技术方案设计；
    - 评审会将在每个任务提交的方案中选择一个最佳方案，确认为本任务的 leading developer，并在开发方向上进行一定的指导，若你的方案为最佳方案，你即可按照完善后的技术方案进行相应开发；

o  **开放性任务**：按照任务要求进行开发即可

- 5、**作品提交：**完成项目开发后，

- - 你需要在相应任务的 Github ISSUE 下以提交      Pull Request 的形式提交你的作品（需包含任务中说明的需要提交的内容）；
  - 并且，在 GitHub 的 [Pinned ISSUE](https://github.com/PaddlePaddle/Paddle/issues/34526) 下，按格式模板回复你作品的 Pull Request 地址，**正式完成作品提交**；

- 6、在作品提交后3个工作日内，评审会将对作品进行验收并给出相应反馈。如果该作品符合要求，会宣布该任务已被完成并更新活动相关信息（**任务被完成后，其他提交将无效**）

## 二、奖项设置

1、常规奖项：根据任务难度，完成相应难度任务并通过评审会评审，将可以获得对应现金奖励。

（2）若该任务仅有一个作品提交，且作品优质，但未100%完成，经评审会评审后，可根据完成进度按照相应比例进行现金奖励发放。

| **⭐ ⭐**   | ￥5,000/题  |

2、特别奖项：在所有提交的作品中，评审团将根据全部作品情况，从创意度、贡献度、实现度三个角度分别给予若干个特别奖项，感谢开发者们对开源社区的贡献：

开源创意奖 ¥3,000

开源贡献奖 ¥3,000

最佳实现奖 ¥3,000

3、其他奖励

- 在活动中，完成作品提交的开发者，均可获得百度开源证书奖励

**特别注意**

1. 以上所有提及金额均为税前金额。
2. 任务奖项评定需开发者提供完整作品（包括任务对应产出内容以及活动总结报名等）及团队成员名单。

## 三、参与方式

1. 活动参与对象

本次活动面向全社会开放，不限年龄、身份、国籍，相关领域的个人、高校、科研机构、企业单位、初创团队等人员均可报名参赛。

**特别注意：**

（1）活动主办、合作项目单位，以及有机会接触任务背景的员工不得参与本次活动；

（2）百度公司员工可参与比赛，但不得领取奖金。

2. 参赛要求

（1）支持以个人或团队的形式报名并认领任务，**每支队伍的人数不超过3人**，允许跨单位自由组队。每人可以认领多个但不超过三个任务。

（2）报名须保证所提供的个人信息真实、准确、有效。如发放奖金或礼品时发现报名人员填写的报名信息与个人身份不相符，组委会将保留停止发放奖金或礼品的权利。

3. 活动报名

（1）2021年9月23日至2021年11月12日，首先在活动官网完成报名。若组队报名，则由队长统一填写报名信息，完成报名即可。

（2）在GitHub 的 [Pinned ISSUE](https://github.com/PaddlePaddle/Paddle/issues/34526) 中，可以查看到本次活动的所有任务，选取感兴趣的任务，按格式回复 队名、任务序号、任务对应 ISSUE 链接，即为完成任务认领；

**补充信息**：

（1）**组队报名方法**：由队长统一填写报名信息即可，包括队员基本信息。也可以个人形式完成报名，完成报名后，联系工作人员进行组队。每队限三人。

（2）官方交流群 【飞桨黑客松QQ交流群】：343734965

4. 作品开发

当你进行相应开发时，相应的代码需要以 Pull Request 的形式上传到任务 ISSUE所在仓库（**代码规范可参考：[代码提交规范](https://github.com/PaddlePaddle/Contrib/wiki/Contrib-代码提交规范)，示例如：[Contrib/PSENet](https://github.com/PaddlePaddle/Contrib/tree/master/PSENet)**），并在 [Pinned ISSUE](https://github.com/PaddlePaddle/Paddle/issues/34526) 中，按格式回复相关的提交信息。评审会一般会在3个工作日内给出作品反馈，如果符合任务要求，则实时通报该任务已经被完成。

**注意**：

（1）活动评审会会**按日更新**所有任务的实时信息到Pinned ISSUE中。

（2）如果发现有恶意提交的现象，可能会取消参与活动资格。

5、飞桨黑客松不收取任何报名费用。

6、在活动过程中有任务问题可以联系我们，发送邮件至 [paddle-hacker@baidu.com](mailto:paddle-hacker@baidu.com) 与我们联系。
