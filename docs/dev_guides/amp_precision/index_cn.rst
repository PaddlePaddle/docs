低精度算子精度保障
############

本章主要介绍飞桨框架中低精度算子实现及其单测开发的规范。

- `低精度算子支持开发规范 <./amp_op_dev_guide_cn.html>`_ : 飞桨低精度算子支持开发规范。
- `低精度算子单测开发规范 <./amp_test_dev_guide_cn.html>`_ : 飞桨低精度算子单元测试完善即添加开发规范。
