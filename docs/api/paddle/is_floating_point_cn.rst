is_floating_point
-------------------------------

.. py:function:: paddle.is_floating_point(x)
判断输入 Tensor 的数据类型是否为浮点类型。

参数
:::::::::

    - **x**  (Tensor) - 输入的 Tensor。

返回
:::::::::

bool，输入 tensor 的数据类型为浮点数类型则为 True，反之为 False.
