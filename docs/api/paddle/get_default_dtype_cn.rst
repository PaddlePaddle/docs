get_default_dtype
-------------------------------

.. py:function:: paddle.get_default_dtype()


得到当前全局的 dtype。该值初始是 float32。


参数
::::::::::::


     无

返回
::::::::::::
 string，这个全局 dtype 仅支持 float16、float32、float64。
