is_complex
-------------------------------

.. py:function:: paddle.is_complex(x)


判断输入 tensor 的数据类型是否为复数类型(complex64 或者 complex128)。

参数
:::::::::
   - **x** (Tensor) - 输入 Tensor


返回
:::::::::
bool，如果输入 tensor 的数据类型为复数类型则为 True，反之为 False
