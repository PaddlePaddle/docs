paddle.distribution
---------------------

paddle.distribution 目录下包含飞桨框架支持的随机变量的概率分布、随机变量的变换、KL 散度相关 API。
具体如下：

-  :ref:`随机变量的概率分布 <about_distribution>`
-  :ref:`随机变量的变换 <about_distribution_transform>`
-  :ref:`KL 散度相关 API <about_distribution_kl>`
