Independent
-------------------------------

.. py:class:: paddle.distribution.Independent(base, reinterpreted_batch_rank)

将一个基础分布 ``base`` 的最右侧 ``reinterpreted_batch_rank`` 批维度转换为事件维度。


参数
:::::::::

- **base** (Distribution) - 基础分布。
- **reinterpreted_batch_rank** (int） - 用于转换为事件维度的批维度数量。
