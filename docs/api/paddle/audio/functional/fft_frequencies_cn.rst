fft_frequencies
-------------------------------

.. py:function:: paddle.audio.functional.fft_frequencies(sr, n_fft, dtype='float32')

计算 fft 频率。

参数
::::::::::::

    - **sr** (int) - 采样率。

返回
:::::::::

``paddle.Tensor``,Tensor shape (n_fft//2 + 1,)。
