set_default_dtype
-------------------------------

.. py:function:: paddle.set_default_dtype(d)


设置默认的全局 dtype。默认的全局 dtype 是 float32。


参数
::::::::::::


     - **d** (string|np.dtype) - 设为默认值的dtype。它仅支持float16、float32和float64。

返回
::::::::::::
 无。
