get_cudnn_version
-------------------------------

.. py:function:: paddle.device.get_cudnn_version()


此函数返回 cudnn 的版本。返回值是 int，它表示 cudnn 版本。例如，如果返回 7600，则表示 cudnn 的版本为 7.6。

返回
::::::::::::
返回一个整数，表示 cudnn 的版本。
