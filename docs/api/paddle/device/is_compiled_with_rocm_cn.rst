is_compiled_with_rocm
-------------------------------

.. py:function:: paddle.device.is_compiled_with_rocm()




检查 ``whl`` 包是否可以被用来在 AMD 或海光 GPU(ROCm)上运行模型。

返回
::::::::::::
bool，支持 GPU(ROCm)则为 True，否则为 False。
