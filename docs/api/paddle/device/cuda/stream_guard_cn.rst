stream_guard
-------------------------------

.. py:function:: paddle.device.cuda.stream_guard(stream)

可以切换当前的 CUDA stream 为输入指定的 stream。

.. note::
    该 API 目前仅支持动态图模式。

参数
::::::::::::

    - **stream** (paddle.device.cuda.Stream) - 指定的 CUDA stream。如果为 None，则不进行 stream 流切换。
