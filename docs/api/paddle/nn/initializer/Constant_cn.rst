Constant
-------------------------------

.. py:class:: paddle.nn.initializer.Constant(value=0.0)




该接口为常量初始化函数，用于权重初始化，通过输入的 value 值初始化输入变量。

参数
::::::::::::

 - **value** (float16|float32，可选) - 用于初始化输入变量的值，默认值为 0。

返回
::::::::::::
参数初始化类的实例。
