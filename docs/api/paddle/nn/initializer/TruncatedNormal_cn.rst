TruncatedNormal
-------------------------------

.. py:class:: paddle.nn.initializer.TruncatedNormal(mean=0.0, std=1.0, name=None)


截断正态分布（高斯分布）初始化方法。

参数
    - **mean** (float，可选) - 正态分布的均值，默认值为 :math:`0.0`。
    - **std** (float，可选) - 正态分布的标准差，默认值为 :math:`1.0`。
    - **name** (str，可选) - 具体用法请参见 :ref:`api_guide_Name`，一般无需设置，默认值为 None。

返回
::::::::::::

    由截断正态分布（高斯分布）初始化的参数。
