ParameterDict
-------------------------------

.. py:class:: paddle.nn.ParameterDict(parameters=None)




参数字典容器。此容器的行为类似于 Python 字典，但它包含的参数将被正确地注册和添加。

参数
:::::::::

    - **parameters** (ParameterDict | Mapping[str, Tensor] | Sequence[tuple[str, Tensor]]，可选) - 可迭代的 Parameters，默认值为 None。

返回
:::::::::
无
