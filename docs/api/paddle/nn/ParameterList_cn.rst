ParameterList
-------------------------------

.. py:class:: paddle.nn.ParameterList(parameters=None)




参数列表容器。此容器的行为类似于 Python 列表，但它包含的参数将被正确地注册和添加。

参数
::::::::::::

    - **parameters** (iterable，可选) - 可迭代的 Parameters。

返回
::::::::::::
无
