CosineSimilarity
-------------------------------
.. py:class:: paddle.nn.CosineSimilarity(axis=1, eps=1e-8)

**CosineSimilarity**

计算 x1 与 x2 沿 axis 维度的余弦相似度。

参数
::::::::::::

  - **axis** (int) - 指定计算的维度，会在该维度上计算余弦相似度，默认值为 1。
  - **eps** (float) - 很小的值，防止计算时分母为 0，默认值为 1e-8。

返回
::::::::::::
无
