tolist
-------------------------------

.. py:function:: paddle.tolist(x)

将 paddle Tensor 转化为 python list。

.. note::
        此 API 仅在 Dygraph 模式下可用。

参数
:::::::::

        - **x** (Tensor) - 输入需要转为 `list`的 `Tensor`。

返回
:::::::::
Tensor 对应结构的 list。
